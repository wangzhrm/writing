比较级和最高级规则

英语中形容词和副词比较级和最高级的构成指南。

1. 单音节形容词和副词

对于大多数单音节词，在词尾加上 -er 构成比较级，加上 -est 构成最高级。

Adjective/Adverb → Adjective/Adverber → Adjective/Adverbest

  • tall (高的) → taller → tallest
  • fast (快的) → faster → fastest

拼写规则: 如果词以“辅音 + 单元音 + 辅音”结尾，双写最后一个辅音字母，再加词尾。

  • big (大的) → bigger → biggest
  • hot (热的) → hotter → hottest

2. 两音节形容词

两音节形容词的规则有所不同。

词尾是 -y:

将 -y 改为 -i，然后加上 -er 或 -est。

  • happy (开心的) → happier → happiest
  • funny (有趣的) → funnier → funniest

其他大部分情况:

在形容词前面加上 more 或 most。

  • famous (著名的) → more famous → most famous
  • boring (无聊的) → more boring → most boring

3. 三个或更多音节的词

对于所有三个或更多音节的词，总是使用 more 和 most。

more + Adjective/Adverb → most + Adjective/Adverb

  • beautiful (美丽的) → more beautiful → most beautiful
  • importantly (重要地) → more importantly → most importantly

4. 不规则词

一些常见的词不遵循上述规则，有自己独特的比较级和最高级形式。

原形 → 比较级 → 最高级

  • good/well (好) → better → best
  • bad/badly (坏/糟糕) → worse → worst
  • little (少) → less → least
  • much/many (多) → more → most
  • far (远) → farther/further → farthest/furthest
