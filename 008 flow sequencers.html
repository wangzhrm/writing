IELTS Writing Task 1: Sequential Linking Phrases for Flow Charts

Sequential Linking Phrases (Sequencers)

These phrases help guide the reader through the sequence and improve the coherence of your report.

Function Expression Examples
Starting the Process The process commences when... / The cycle begins with... / The first step involves...
Moving to the Next Step Following this, the material... / Once X is complete... / Subsequently, Y is... / In the next stage... / At this point...
Showing Simultaneous Steps While X is being processed... / Simultaneously, another pipe... / At the same time...
Concluding the Process The final step is when... / Finally... / Ultimately... / The cycle concludes with...

Example Sentences for Sequencers

1. Starting the Process: The cycle begins with...

The cycle begins with the collection of raw materials from the forest.

循环从森林中收集原材料开始。

2. Moving to the Next Step: Following this, the material...

Following this, the material is transported to the processing plant.

在此之后，材料被运输到加工厂。

3. Showing Simultaneous Steps: At the same time...

At the same time, quality checks are performed on the processed goods.

与此同时，对加工好的产品进行质量检查。

4. Concluding the Process: Finally...

Finally, the finished products are packaged for distribution.

最后，成品被包装以供分发。
