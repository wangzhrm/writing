example sentences

1. Produce / Produced (生产 / 被生产)

Paper is produced from wood pulp in the factory.

纸张在工厂中由木浆生产。

2. Collect / Collected (收集 / 被收集)

Raw materials are collected from the forest.

原材料从森林中被收集。

3. Process / Processed (处理 / 被处理)

The pulp is processed into a smooth mixture.

纸浆被处理成均匀的混合物。

4. Transport / Transported (运输 / 被运输)

Materials are transported to the processing plant.

材料被运输到加工厂。

5. Store / Stored (储存 / 被储存)

Finished products are stored in the warehouse.

成品被储存在仓库中。

6. Package / Packaged (包装 / 被包装)

Products are packaged in boxes for shipping.

产品被包装在盒子中以便运输。

7. Distribute / Distributed (分发 / 被分发)

Products are distributed to retailers nationwide.

产品被分发到全国的零售商。

8. Assemble / Assembled (组装 / 被组装)

Parts are assembled into the final product.

零件被组装成最终产品。

9. Extract / Extracted (提取 / 被提取)

Minerals are extracted from the earth.

矿物从地球中被提取。

10. Treat / Treated (处理 / 被处理)

Water is treated to remove impurities.

水被处理以去除杂质。

11. Heat / Heated (加热 / 被加热)

The mixture is heated to a high temperature.

混合物被加热到高温。

12. Cool / Cooled (冷却 / 被冷却)

The liquid is cooled to form a solid product.

液体被冷却以形成固体产品。

13. Mix / Mixed (混合 / 被混合)

Ingredients are mixed in a large container.

原料在大型容器中被混合。

14. Test / Tested (测试 / 被测试)

Products are tested for quality standards.

产品被测试以确保质量标准。

15. Recycle / Recycled (回收 / 被回收)

Plastic waste is recycled into new products.

塑料废物被回收成新产品。

16. Turn into / Turned into (变成 / 被变成)

Raw materials are turned into pulp during processing.

原材料在加工过程中被变成纸浆。

17. Form / Formed (形成 / 被形成)

Sheets are formed from the processed pulp.

纸张从加工过的纸浆中被形成。

18. Generate / Generated (产生 / 被产生)

Energy is generated from waste materials.

能源从废料中被产生。

19. Create / Created (创造 / 被创造)

New products are created from recycled materials.

新产品从回收材料中被创造。

20. Develop / Developed (开发 / 被开发)

Prototypes are developed during the design phase.

原型在设计阶段被开发。

21. Crush / Crushed (压碎 / 被压碎)

Raw materials are crushed into smaller pieces.

原材料被压碎成小块。

22. Grind / Ground (研磨 / 被研磨)

Grains are ground into fine powder.

谷物被研磨成细粉。

23. Separate / Separated (分离 / 被分离)

Components are separated during recycling.

在回收过程中，部件被分离。

24. Filter / Filtered (过滤 / 被过滤)

Liquid is filtered to remove impurities.

液体被过滤以去除杂质。

25. Purify / Purified (净化 / 被净化)

Water is purified before bottling.

水在装瓶前被净化。

26. Melt / Melted (熔化 / 被熔化)

Plastic is melted for reshaping.

塑料被熔化以便重新塑造。

27. Boil / Boiled (煮沸 / 被煮沸)

Water is boiled to eliminate bacteria.

水被煮沸以消除细菌。

28. Condense / Condensed (冷凝 / 被冷凝)

Vapor is condensed into liquid form.

蒸汽被冷凝成液体形式。

29. Evaporate / Evaporated (蒸发 / 被蒸发)

Liquid is evaporated to concentrate the solution.

液体被蒸发以浓缩溶液。

30. Inject / Injected (注入 / 被注入)

Additives are injected into the mixture.

添加剂被注入混合物中。

31. Pump / Pumped (泵送 / 被泵送)

Water is pumped into storage tanks.

水被泵送至储罐中。

32. Feed / Fed (供给 / 被供给)

Raw materials are fed into the machine.

原材料被供给到机器中。

33. Stretch / Stretched (拉伸 / 被拉伸)

Plastic film is stretched to form sheets.

塑料薄膜被拉伸以形成薄片。

34. Flatten / Flattened (压平 / 被压平)

Metal sheets are flattened for further processing.

金属板被压平以便进一步加工。

35. Roll / Rolled (滚动 / 被滚动)

Paper is rolled into large cylinders.

纸张被卷成大圆筒。

36. Examine / Examined (检查 / 被检查)

Products are examined for defects.

产品被检查以发现缺陷。

37. Inspect / Inspected (检验 / 被检验)

Items are inspected before packaging.

物品在包装前被检验。

38. Measure / Measured (测量 / 被测量)

Dimensions are measured for accuracy.

尺寸被测量以确保准确性。

39. Monitor / Monitored (监控 / 被监控)

The process is monitored for consistency.

过程被监控以确保一致性。

40. Load / Loaded (装载 / 被装载)

Goods are loaded onto trucks.

货物被装载到卡车上。

41. Unload / Unloaded (卸载 / 被卸载)

Materials are unloaded at the factory.

材料在工厂被卸载。

42. Deliver / Delivered (交付 / 被交付)

Products are delivered to retailers.

产品被交付给零售商。

43. Send / Sent (发送 / 被发送)

Packages are sent to distribution centers.

包裹被发送到分发中心。

44. Remove / Removed (移除 / 被移除)

Impurities are removed from the liquid.

杂质从液体中被移除。

45. Eliminate / Eliminated (消除 / 被消除)

Contaminants are eliminated during filtration.

在过滤过程中，污染物被消除。

46. Blend / Blended (混合 / 被混合)

Ingredients are blended to create a uniform mixture.

原料被混合以创建均匀的混合物。

47. Add / Added (添加 / 被添加)

Chemicals are added to the solution.

化学物质被添加到溶液中。

48. Combine / Combined (结合 / 被结合)

Materials are combined in a mixing chamber.

材料在混合室中被结合。
